## Unique Angle for PS1 (jisse tum stand out karoge)

Most teams banayenge: "to-do list + reminder + calendar sync." Boring, generic.

Tumhara unique twist ho sakta hai: **"Commitment Contract Agent"**

1. **Procrastination Detector** — agent track kare ki user ne task start kiya ya nahi (check-ins ke through), aur agar delay dikhe to apne se escalate kare — pehle gentle nudge, fir harder consequence-based nudge (jaise "agar 2 ghante mein nahi kiya, main tumhare mentor/friend ko bata dunga" — with user consent obviously).

2. **Micro-task Breakdown Agent** — bada task (jaise "assignment likhna") ko khud break kare chhote 15-20 min ke actionable steps mein, aur ek step complete hone par next step proactively suggest kare.

3. **Context-aware reasoning** — agent sirf time-based reminder na de, balki user ke calendar, energy pattern, aur past behavior se decide kare "abhi karna best time hai ya nahi."

4. **Voice-based check-in** — Gemini ke voice capability se daily 2-min voice check-in lena: "aaj kya pending hai, kya blocker hai" — aur usi se plan update karna.

5. **Autonomous email/message drafting** — agar deadline miss hone wali hai aur extension chahiye, agent khud ek polite email draft kare professor/manager ko (user approval ke baad bhejna).

Yeh sab combine karke ek "agent jo sirf remind nahi karta, balki tumhare saath kaam complete karwata hai" — judges ke liye ye fresh aur "agentic" lagega.

## 1. Core Concept

Ek AI agent jo sirf reminder nahi deta — balki:
- User ke tasks samajhta hai
- Unko chhote actionable steps mein todta hai
- Proactively check-in karta hai
- Procrastination detect karta hai aur escalate karta hai
- Khud kuch actions le sakta hai (draft emails, reschedule, suggest next step)

---

## 2. Detailed Workflow (User Journey)

### Step A — Onboarding
1. User sign up/login karta hai (simple auth)
2. User apna goal/task add karta hai — text se ya **voice se** (e.g., "Mujhe Friday tak ek assignment submit karna hai, abhi shuru nahi kiya")
3. Gemini API task ko parse karta hai → deadline, priority, category extract karta hai

### Step B — AI Task Breakdown
4. Gemini agent bade task ko **micro-tasks** mein break karta hai (e.g., "Research" → "Outline" → "Draft" → "Review" → "Submit")
5. Har micro-task ko ek estimated time aur suggested schedule slot milta hai (calendar ke free slots dekh ke)

### Step C — Smart Scheduling
6. Agent Google Calendar se sync karta hai (Calendar API), free time slots dhundta hai
7. Micro-tasks ko calendar mein auto-suggest karta hai ("Kal 4-5pm free ho, outline yahan kar lo?")
8. User approve/reject/reschedule kar sakta hai

### Step D — Proactive Check-ins (Yeh agentic part hai)
9. Scheduled time pe agent check-in karta hai: "Outline start kiya?" 
10. Agar user "haan" bole → next step unlock, progress track
11. Agar user "nahi" bole ya respond na kare → **escalation logic** trigger:
    - Pehla nudge: friendly reminder
    - Doosra nudge: deadline ka pressure highlight ("Sirf 6 ghante bache hain")
    - Teesra: agent suggest kare ki kya chhod sakte ho ya extension maango

### Step E — Autonomous Assistance
12. Agar deadline risk mein hai, agent khud offer kare:
    - "Chaaho to main professor/manager ko ek polite extension email draft kar dun?"
    - User approve kare to Gemini email draft generate karega (auto-send nahi, user review karega — safety ke liye)
13. Agent daily/weekly summary deta hai: kitna complete hua, kya pending hai

### Step F — Voice Interaction (optional but impressive)
14. User voice se bol sakta hai: "Aaj kya pending hai?" → agent voice/text mein jawab de

5. **Escalation Engine** — procrastination detect → nudges intensify hote hain
6. **Autonomous Drafting** — extension emails, follow-ups khud draft karna
7. **Voice-enabled Assistant** — voice commands aur replies

## 4. Tech Stack (Detailed)

### Core AI
- **Google Gemini API** (via Google AI Studio) — task parsing, breakdown, conversational agent, function calling/tool use
- **Gemini Function Calling** — agent ko tools dena (e.g., `create_calendar_event`, `draft_email`, `update_task_status`)

### Frontend
- **React.js** (ya simple HTML/CSS/JS agar time kam hai) — chat-like UI + dashboard
- **Tailwind CSS** — styling fast karne ke liye

### Backend
- **Node.js + Express** (ya Python Flask/FastAPI) — API layer jo Gemini ko calls kare, logic handle kare
- **Firebase** (Firestore) — user data, tasks, schedules store karne ke liye (easy aur free tier achha hai)

### Google Technologies (important for "Google Tech" criteria - 15%)
- **Google AI Studio** — core build/deploy platform (mandatory)
- **Gemini API** — main intelligence
- **Google Calendar API** — scheduling integration
- **Google OAuth** — login
- (Optional) **Web Speech API / Gemini multimodal voice** — voice input/output

### Hosting/Deployment
- **Google AI Studio's built-in deploy feature** (jo guidelines mein mandatory hai)
- GitHub for code repo

### Scheduling/Automation (agar time mile)
- Simple **cron-like scheduler** (node-cron) backend mein for check-in reminders
